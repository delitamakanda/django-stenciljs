# django-angular
Experimental app in angular

![Python package](https://github.com/delitamakanda/dating-app/workflows/Python%20package/badge.svg?branch=master)

## Frontend

```bash
npm -g install @angular/cli
ng --version

ng new app
```

```bash
cd frontend && ng serve or npm run start
```
